# PadelNexo

Base inicial creada con Expo y React Navigation.

## 1. Crear el proyecto con Expo

```bash
npx create-expo-app@latest padelnexo-app --template blank
cd padelnexo-app
```

## 2. Instalar navegación

```bash
npm install @react-navigation/native @react-navigation/native-stack
npx expo install react-native-screens react-native-safe-area-context
```

## 3. Reemplazar los archivos

Copiar la estructura de este proyecto dentro de la carpeta creada por Expo.

## 4. Ejecutar la app

```bash
npx expo start
```

## Variables de entorno

Para pruebas de Mercado Pago Checkout Pro en frontend:

```bash
EXPO_PUBLIC_MERCADO_PAGO_PUBLIC_KEY=APP_USR-xxxxxxxxxxxxxxxxxxxxxxxx
```

## 5. Abrir en Expo Go

- Instalar Expo Go en el celular.
- Conectar el celular y la computadora a la misma red Wi-Fi.
- Escanear el codigo QR que aparece en la terminal o en el navegador.

## Pantallas incluidas

- Home
- Login
- Registro
